Um an einem Beispiel die Vorteile von xmera Omnia zeigen zu können, wurde die demo GmbH erfunden. Die demo GmbH ist nur so weit erdacht, dass alle Prozesse eines ISMS dargestellt werden können.

== Produkte

Die demo GmbH bietet den Kunden IT-Dienstleistungen an. In diesem Zusammenhang wird auch Software programmiert und für die Tests und Migrationsprojekte Daten der Kunden verarbeitet.

== Mitarbeiter

- Geschäftsführung,
- Verwaltung,

== Prozesse

In der demo GmbH werden im ISMS

- die Abrechnung,
- die Angebotserstellung,
- das Personalmanagement,
- die Projektdurchführung und
- die Projektgewinnung

betrachtet.

== Netzstrukturplan

Um in der xmera Omnia Demo auch ein IT-Assetregister darstellen zu können, wurde auch eine fiktive IT-Struktur erdacht. Die Struktur ist in dem folgenden Netzstrukturplan abgebildet.

image::einfuehrung_demo-gmbh_netzstruktur.png[Netzstrukturplan,600,800,align=left]

Die Netzstruktur zeigt eine Hybride IT-Umgebung aus einer eigenen IT für die Entwicklung und die Verwaltung, aber auch die Nutzung von Cloud-Diensten ist abgebildet.

Die demo GmbH hat einen Standort. Die Mitarbeiter können aber auch aus dem Home-Office arbeiten.
